
## Description

The projects implements a Czech National Bank Currency Converter.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run dev

```

## Run tests

```bash
# unit tests
$ npm run test
```
